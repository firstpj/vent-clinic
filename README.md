# 📖 [Vet clinic databse] <a name="about-project"></a>

**[Vet clinic database]** This project utilizes a relational database to handle the data and information associated with a veterinary clinic. PostgreSQL serves as the chosen database management system to both create and execute queries on tables within the database.
